# blinkit-sales-dashboard
Blinkit Dashboard
An interactive sales analytics dashboard built using Power BI to analyze Blinkit grocery store performance across sales, outlet operations, customer preferences, and product categories.

This project was created as a guided learning project by following a Power BI tutorial from the YouTube channel Data Tutorials while implementing the complete dashboard independently for hands-on practice and skill development.

The project helped strengthen practical understanding of:

Data cleaning
Power Query transformations
DAX calculations
Dashboard design
Data visualization
Retail business analytics

The dashboard provides insights into:

✅ Total Sales Performance
✅ Average Sales & Product Ratings
✅ Outlet Size & Location Analysis
✅ Product Category Performance

🔍 Key Learning & Implementation Areas
✅ Data Cleaning & Transformation (Power Query)
Performed multiple preprocessing steps including:

Handling missing values
Category-wise median imputation for missing item weights
Data type corrections
Data transformation
Data formatting and preprocessing
One of the major learning outcomes was understanding how to use Power Query for structured data preparation before visualization.

✅ DAX Measures & KPI Development
Created measures for:

Total Sales = SUM('BlinkIT Grocery Data'[Sales])
Avg Sales = AVERAGE('BlinkIT Grocery Data'[Sales])
No of Items = COUNT('BlinkIT Grocery Data'[Item Identifier])
Avg Rating = AVERAGE('BlinkIT Grocery Data'[Rating])

📈 Dashboard Insights
Some key observations from the dashboard:

Tier 3 outlets generated the highest sales contribution.
Medium-sized outlets contributed the largest revenue share.
Fruits & Vegetables and Snack Foods were among the highest-performing item categories.
Outlet establishments around 2018 showed strong sales performance trends.
Different outlet tiers showed varying performance across low-fat and regular product categories.
